# news-keyword-poc

소상공인 뉴스 키워드 트렌드 PoC. 목표는 (1) 임베딩 기반 무관 뉴스 필터링, (2) 키워드 추출 + 급상승 스코어의 가능성 검증.

## 원칙
- PoC 범위를 넘는 기능(대시보드, 실시간화, fine-tuning, NER)은 제안만 하고 구현하지 않는다
- 모든 하이퍼파라미터는 config.yaml에서 관리, 코드에 하드코딩 금지
- 데이터는 제목+요약문만 사용, 본문 크롤링 금지
- 단계별 스크립트는 독립 실행 가능해야 하며, 중간 산출물은 data/에 저장
- 성공 기준: 필터 precision ≥ 85%, 상위 20개 키워드 중 관련 키워드 ≥ 15개

## 실행 순서
- 엔진 A (뉴스, 정책·사업환경 트렌드): collect → filter → extract → trend → evaluate
- 엔진 B (블로그, 소비 유행/신조어): buzz_collect → buzz_extract
- 두 엔진은 목적·소스가 다르다. 뉴스는 유행의 후행 지표라 소비 신조어는 엔진 B(soynlp 통계 추출)가 담당한다.

## 스타일
- 함수에 타입 힌트, 핵심 로직에 한국어 주석
- 외부 API 호출은 재시도 + rate limit 준수 (네이버 API: 초당 10회 제한)
