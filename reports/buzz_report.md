# 엔진 B PoC 결과 리포트: 블로그 기반 신조어/소비 유행 발굴

생성일: 2026-07-02 / 입력: 네이버 블로그 4,799건 (최근 4주)

## 목표

뉴스는 유행의 후행 지표라 소비 유행어(예: "두쫀쿠")를 놓친다. 블로그에서 **형태소 사전에 의존하지 않는 통계 기반 추출**로 신조어/유행어를 자력 발굴할 수 있는지 검증한다.

## 방법

1. **수집** (`buzz_collect.py`): 네이버 블로그 검색 API, 소비 트렌드 쿼리("요즘 유행 디저트", "신상 카페", "핫플 맛집" 등), 제목+요약문만
2. **단어 추출** (`buzz_extract.py`): soynlp `WordExtractor`로 사전 없이 응집도(cohesion) 높은 단어를 추출 → `MaxScoreTokenizer`로 포스트를 단어 단위 분해(조사 자연 분리)
3. **정제**: 응집도·길이·불용어 + 품사(모든 토큰 명사) + 파편 제거(끝 조사글자 필터 + 접두 파편 dedup: 디저트맛→디저트맛집)
4. **단어 유형 분류** (kiwipiepy `.oov` = 사전 미등재 여부 기반):
   - **신조어**: OOV 형태소 포함 → 왁뿌·탕후루·두쫀쿠·휘낭시에·빵지순례 (kiwi가 모르는 진짜 신생어)
   - **합성어**: 기존어 결합 → 소금빵·흑돼지·오션뷰 (신조어가 아니라 기존 단어 조합)
   - **일반어**: 사전 단일어 → 디저트·두바이
   - (이전 버전은 "단일 명사로 인식 못하면 신조어"로 정의해 소금빵을 신조어로 오분류하고 왁뿌를 놓쳤음 → `.oov`로 교정)
5. **급상승 스코어**: 엔진 A와 동일한 주 단위 `이번 주 / (이전 3주 평균 + 1)`

## 한계 및 원인

- **블로그 광고/메타 노이즈**: "서이추환영"(서로이웃 추가), 업소 정보("운영시간", "정기휴무") 등이 후보에 섞여 불용어로 일부 제거. 체험단·협찬 포스팅의 상업적 편향은 남아 있음
- **지명·업태 일반어 혼입**: "선릉역", "가로수길", "무한리필"처럼 신조어는 아니지만 트렌드 신호인 것과, "국내여행" 같은 일반어가 공존 → 신조어 플래그는 참고용, 최종 선별은 수동 검토 필요
- **트렌드 baseline 부족**: 4주 수집 + 대상 주(W27) 편중으로 급상승 스코어가 부풀려짐. 신조어는 "과거 0 → 이번 주 급증" 패턴이라 스코어 공식과는 잘 맞지만, 안정적 추세 판단에는 정기 수집 누적이 필요
- **응집도 임계값 민감**: min_cohesion/min_frequency에 따라 후보 수가 크게 변함 (config에서 조정)

## 향후 확장

- 발굴된 후보를 **네이버 데이터랩 검색어트렌드 API로 교차 검증**(발굴→검증 파이프라인 완성)
- 광고/체험단 포스팅 분류기로 상업적 편향 제거
- 인스타그램·유튜브 등 소스 다변화
