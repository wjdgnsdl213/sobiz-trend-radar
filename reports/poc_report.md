- 신조어 미인식(형태소 분석기 사전 한계) — 소비 유행어(예: 신메뉴명)는 뉴스+형태소 조합으로 잘 안 잡힘. 블로그+통계 기반 추출(엔진 B)로 별도 검증 예정
- near-duplicate 182건을 임베딩 유사도 0.92로 제거했으나, 패러프레이즈된 보도자료(0.92 미만)는 일부 잔존 — 상위 키워드에 특정 기업 홍보 이벤트(예: 우리금융 미소금융)가 남을 수 있음
- 제목+요약문만 사용(본문 크롤링 금지) — 본문에만 있는 키워드 누락
- 트렌드 스코어는 대상 주 문서 편중(API 1,000건 상한)으로 부풀려짐 — 정기 수집 누적으로 해소
- 과거 데이터 확보에는 빅카인즈 API 등 대체 소스 필요
- 향후 확장(범위 밖): 대시보드, 실시간 수집 스케줄러, 분류기 fine-tuning, NER 기반 지역×음식 분석, DB 연동
